git解除子模块的引用

在git管理的项目中经常使用第三方git库或资源，而由于各种原因这些git库或资源(下文称为子模块)需要进行修改才能达到我们希望的效果。
这些子模块将会完成。
依次执行如下命令，即可解除。

git rm —cached submodule_path # 删除子模块引用头(没有斜杠号)
git rm .gitmodules # 如有一个以上的子模块, 需要编辑这个文件,而不是删除!
rm -rf submodule_path/.git # 确保你有备份!!
git add submodule_path # 添加文件,而不是提交参考
git commit -m “remove submodule”

参考 un-submodule a git submodule 。
